# Backbase Assist (Employee Experience)

## Licensing Model
- **Primary metric:** Per employee (bank staff) per year
- Covers front-line staff, relationship managers, call center agents

## Pricing Reference Ranges
- **Standard (call center/branch):** $200 – $500 per employee/year
- **Advanced (RM/wealth advisor):** $500 – $1,200 per employee/year
- **Enterprise (full suite):** $300 – $800 per employee/year (volume discounts)

## Common Bundling Patterns
- Often bundled with Digital Banking for "full experience" positioning
- Wealth management: Assist is core to RM productivity story
- Can be standalone for employee-only transformation plays

## Key Commercial Considerations
- Employee count is more stable than user count — predictable ARR
- Staff reduction narrative: client may resist per-employee if headcount is shrinking
- Position as "per advisor" for wealth (higher perceived value)
