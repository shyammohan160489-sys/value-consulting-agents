# Backbase 2025 Pricing Model — License Editions

## New Pricing Structure (2025)
The 2025 model replaces the old fluctuating base fee + per-user model with:
- **Fixed Base Fee** with first user tier included in the price
- **Managed Hosting embedded** into the Base Fee
- **Additional Users** as the ONLY discountable component

## Retail Banking Editions

### Essential Edition
- **Features:** Dashboard, Accounts, Transactions, Statements, Payments, Contacts, Card Management, Loan Management, Notifications, Message Centre, My Profile, Identity Essential, Assist Essential, Engage Essential, QR Payments
- **Base Fee:** $494,400/year
- **Included Users:** 20,000
- **Additional User Rates (tiered):** $6.52 → $1.02/user (decreasing tiers)

### Premium Edition
- **Features:** All Essential + TVP Multi Segment, Personal Dashboard, Advanced Card Management, Budgets, Spend Insights, Shopping Center, Virtual Cards, Service Centre, Pockets
- **Base Fee:** $607,360/year
- **Included Users:** 20,000
- **Additional User Rates (tiered):** $9.52 → $1.39/user (decreasing tiers)

### Signature Edition
- **Features:** All Premium + Family Banking, Household Banking (Shared Accounts), Financial Advice (Beta)
- **Base Fee:** $786,800/year
- **Included Users:** 20,000
- **Additional User Rates (tiered):** $13.12 → $1.84/user (decreasing tiers)

## Key Commercial Implications
- **Only the Additional User rate is discountable** — base fee is fixed
- Hosting is no longer a separate line item (embedded in base fee)
- Editions create natural upsell path: Essential → Premium → Signature
- Per-user cost decreases as user count grows (tiered) — good client optics
- Regional rollout model applies across editions
